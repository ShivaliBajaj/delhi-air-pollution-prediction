# Delhi COPD Risk Analysis

Early detection of COPD risk using air pollution exposure
and medical imaging data, focused on Delhi NCR.

Project structure finalized and baseline setup completed.

## Why this project exists

COPD is not an abstract medical problem in India — it’s lived, chronic, and deeply urban.
Delhi, in particular, sits at the intersection of air quality, population density,
and long-term respiratory health outcomes.

This project is my attempt to study COPD in Delhi using real-world data from
reputed public and research sources (2024–2026), and to answer a very practical question:

**Does this problem even deserve machine learning — or would simpler, transparent
analysis be more honest and useful?**

The focus here is not model worship.
It is clarity, reasoning, and solution-oriented thinking.
